# 🥔 Micro Tato — play in your browser

A Brotato-style nanobot arena-survivor. **[▶ Play here](https://4444j99.github.io/micro-tato-play/)**

Works on desktop and phone (touch controls appear automatically). Made in Godot 4.7.
This repo holds only the compiled web build; the source lives elsewhere.

Controls (desktop): move WASD / arrows · Shift dash · Space special · F limit · V spectate.
